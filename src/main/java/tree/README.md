# 二叉树刷题顺序

建议按下面顺序刷，从易到难推进。每一题先自己写，卡住再看已有代码或提示。

## 1. 基础递归

1. `basic/Main104二叉树的最大深度.java` - Easy
   - 练习递归终止条件和左右子树结果合并。

2. `basic/Main226翻转二叉树.java` - Easy
   - 练习递归处理左右子树，以及交换左右孩子。

3. `basic/Main101对称二叉树.java` - Easy
   - 练习两个节点同时递归比较。

4. `basic/Main111二叉树的最小深度.java` - Easy
   - 练习空子树边界，注意只有一个子树时不能直接取 0。

## 2. 遍历

5. `traversal/Main102二叉树的层次遍历.java` - Medium
   - 练习 BFS 队列模板。

## 3. 路径题

6. `path/Main112路径总和.java` - Easy
   - 练习根到叶子路径判断。

7. `path/Main113路径总和II.java` - Medium
   - 练习回溯，路径需要加入和撤销。

8. `path/Main129二叉树根节点到叶子节点的所有路径和.java` - Medium
   - 练习把路径上的节点值累积成状态。

## 4. 二叉搜索树

9. `bst/Main98验证二叉搜索树.java` - Medium
   - 练习上下界递归，不能只比较当前节点和左右孩子。

10. `bst/二叉搜索树.java` - 模板练习
    - 练习 BST 查找、插入、删除等基本操作。

11. `bst/Main235二叉搜索树的最近公共祖先.java` - Medium
    - 利用 BST 有序性质找最近公共祖先。

## 5. 最近公共祖先

12. `lca/Main236二叉树的最近公共祖先.java` - Medium
    - 普通二叉树没有有序性质，要靠左右子树递归返回结果。

## 6. 构造二叉树

13. `build/Main重建二叉树.java` - 模板练习
    - 先用已有写法理解前序 + 中序重建树。

14. `build/Offer7重建二叉树.java` - Medium
    - 同类题，可以对比不同写法。

15. `build/Main105从前序与中序遍历序列构造二叉树.java` - Medium
    - 按 LeetCode 标准方法签名重新写一遍，建议用 HashMap 优化中序下标查找。

## 7. 树形 DP

16. `dp/Main543二叉树的直径.java` - Easy
    - 入门树形 DP：递归返回深度，同时更新全局答案。

17. `dp/Main337打家劫舍III.java` - Medium
    - 练习“选当前节点”和“不选当前节点”的状态设计。

18. `dp/Main124二叉树中的最大路径和.java` - Hard
    - 难题。递归返回单边最大贡献，全局答案可以同时取左右贡献。

## 推荐复习节奏

第一遍重点是写出来，不追求最优雅。

第二遍按专题复盘：

1. 递归函数的返回值是什么。
2. 当前节点要做什么。
3. 是否需要全局变量。
4. 是否需要回溯撤销状态。
5. 空节点、叶子节点、单边子树怎么处理。
