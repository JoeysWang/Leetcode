## 1. Shell：词频统计

```bash
cat words.txt | tr -s ' ' '\n' | sort | uniq -c | sort -nr | awk '{print $2" "$1}'
```

```bash
grep -E '^(\([0-9]{3}\) [0-9]{3}-[0-9]{4}|[0-9]{3}-[0-9]{3}-[0-9]{4})$' file.txt
```

## 3. Shell：转置文件

```bash
awk '{ for (i = 1; i <= NF; i++) a[NR, i] = $i } NF > max { max = NF } END { for (i = 1; i <= max; i++) { line = a[1, i]; for (j = 2; j <= NR; j++) line = line " " a[j, i]; print line } }' file.txt
```


## 7. SQL：员工收入超过经理

```sql
select e.Name as Employee
from Employee e join Employee m on e.ManagerId = m.Id
where e.Salary > m.Salary;
```


## 11. SQL：上升的温度

```sql
select w1.Id
from Weather w1 join Weather w2
on datediff(w1.RecordDate, w2.RecordDate) = 1
where w1.Temperature > w2.Temperature;
```

## 12. 10. Regular Expression Matching

'.' 匹配任意单字符，'*' 匹配前一个字符零次或多次。

```java
public boolean regexMatch(String s, String p) {
    int m = s.length();
    int n = p.length();
    boolean[][] dp = new boolean[m + 1][n + 1];
    dp[0][0] = true;
    for (int j = 2; j <= n; j++) {
        if (p.charAt(j - 1) == '*') {
            dp[0][j] = dp[0][j - 2];
        }
    }

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            char pc = p.charAt(j - 1);
            if (pc == '*') {
                dp[i][j] = dp[i][j - 2];
                char prev = p.charAt(j - 2);
                if (prev == '.' || prev == s.charAt(i - 1)) {
                    dp[i][j] |= dp[i - 1][j];
                }
            } else if (pc == '.' || pc == s.charAt(i - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            }
        }
    }
    return dp[m][n];
}
```

## 13. 44. Wildcard Matching

'?' 匹配单字符，'*' 匹配任意长度字符串。

```java
public boolean wildcardMatch(String s, String p) {
    int m = s.length();
    int n = p.length();
    boolean[][] dp = new boolean[m + 1][n + 1];
    dp[0][0] = true;
    for (int j = 1; j <= n; j++) {
        if (p.charAt(j - 1) == '*') {
            dp[0][j] = dp[0][j - 1];
        }
    }

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            char pc = p.charAt(j - 1);
            if (pc == '*') {
                dp[i][j] = dp[i][j - 1] || dp[i - 1][j];
            } else if (pc == '?' || pc == s.charAt(i - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            }
        }
    }
    return dp[m][n];
}
```

## 14. 概念：进程、线程、文件描述符

- **进程**：资源分配单位，拥有独立地址空间
- **线程**：CPU 调度单位，多个线程共享同一进程资源
- **文件描述符**：进程打开文件、socket、管道等内核对象后得到的整数句柄

---

## 15. 概念：Cookie、Session、Token

- **Cookie**：存在客户端，会随请求发送
- **Session**：通常存在服务端，客户端只保存 session id
- **Token**：可携带身份信息或签名的令牌，服务端通过校验 token 判断身份

---

## 16. 概念：加密基础

- **哈希**：不可逆摘要，常用于完整性校验和密码摘要
- **对称加密**：加解密用同一密钥，速度快
- **非对称加密**：有公钥和私钥，适合密钥交换和身份认证
- **签名**：用私钥生成、公钥验证，证明来源和防篡改
